Fetching Data with Fetch API

February 16, 2025

I decided to dynamically load HTML content into my site using the fetch() API. When a header link is clicked, the corresponding HTML page is fetched /pages/${pageId}.html and the content is inserted into body. After the content is loaded, syntax highlighting is reapplied to any code blocks within the new content.


fetch(`/pages/${pageId}.html`).then((response) => response.text()).then((data) => {
    document.getElementById("page-header").innerHTML = `> ${pageId}`;
    document.getElementById("main-content").innerHTML = data;
});
